# 제품 카탈로그 디렉토리

이 폴더에 PDF 카탈로그 파일을 넣어주세요.

## 필요한 파일

다음 파일명으로 PDF를 추가하세요:

1. **ecosa-catalog-2025.pdf** (에코사 종합 카탈로그)
2. **ecosh-catalog.pdf** (에코쉬 무수소변기 카탈로그)
3. **toilet-catalog.pdf** (절수형 양변기 카탈로그)
4. **neoamico-catalog.pdf** (NEOAMICO 카탈로그)

## 사용 방법

1. PDF 파일을 이 폴더(`public/catalogs/`)에 복사
2. 파일명이 위와 정확히 일치하는지 확인
3. 개발 서버가 실행 중이면 자동으로 반영됩니다

## 표시 위치

카탈로그 다운로드 버튼은 다음 위치에 표시됩니다:

- **제품 소개 섹션 상단**: 4개의 카탈로그 카드
- **각 제품 상세 정보**: 제품별 카탈로그 다운로드 버튼
- **CTA 섹션**: "카탈로그 다운로드" 항목

## 파일 크기

현재 설정된 파일 크기 (실제 파일과 다를 수 있음):
- 종합 카탈로그: 5.2MB
- 에코쉬: 2.8MB
- 절수형 양변기: 1.5MB
- NEOAMICO: 2.1MB

파일 크기를 수정하려면 `src/content/site.ts`의 `catalogs` 섹션을 편집하세요.
